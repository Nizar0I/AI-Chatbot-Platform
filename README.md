# 🤖  Chatbot Platform

Full-stack AI chatbot web application developed with React.js (frontend) and Laravel (backend).

## 🚀 Features

- Real-time chatbot interaction
- REST API integration
- Responsive user interface
- Conversation management

## 🛠️ Technologies

- React.js
- Laravel
- MySQL
- REST API

## 📁 Structure

backend de chatbot/
interface de chatbot/

## ⚙️ Installation

### Backend
cd backend de chatbot
composer install
cp .env.example .env
php artisan key:generate
php artisan serve

### Frontend
cd interface de chatbot
npm install
npm run dev
